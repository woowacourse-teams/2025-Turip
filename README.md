# ✈️🧑‍🤝‍🧑 튜립 (Turip)

> 유튜버의 여행 정보를 한눈에 빠르게 확인할 수 있는 서비스

## 📝 프로젝트 소개
> 이 서비스는 유튜버가 실제 다녀온 여행 동선, 장소를 수집하고 영상과 함께 제공하는 앱입니다.  
사용자는 지역을 기준으로 여행 루트를 검색할 수 있으며, 원하는 장소를 내 폴더에 찜! 할 수 있습니다.
친구, 가족, 연인과 함께 떠날 장소를 모아서 공유까지 해보세요! 💙
